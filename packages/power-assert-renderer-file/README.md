Produces filepath and line number of assertion.

```
  # test/some_test.js:26
  assert(a.name === 'bar')
```


USAGE
---------------------------------------

`var FileRenderer = require('power-assert-renderer-file');`



INSTALL
---------------------------------------

```sh
$ npm install --save-dev power-assert-renderer-file
```
